This searcher takes in regex and then searches through the files uploaded and produces a file containing these search results. You can specify what you want to return by adding brackets. For example: "Hello (World)" will search for "Hello World"s, and return a list of World. A more sophisticated example would be: "Hello ([A-Za-z])" which will search for "Hello [NAME]" e.g. "Hello Dolan", and then return a list of names.
